# 快速开始

提示

该组件只可以在 uniapp 项目中使用。

npm i press-swiper

我们更推荐在引入时，使用 uniapp 的easycom (opens new window)方式引入。

// pages.json

{
    "easycom": {
        "^z-(.*)": "press-swiper/components/z-$1/z-$1.vue"
    },
    "pages": [...],
    "globalStyle": {
        ...
    }
}
