# 快速开始

:::warning 提示
该组件只可以在 uniapp 项目中使用。
:::


```bash
npm i press-swiper
```

我们更推荐在引入时，使用 uniapp 的[easycom](https://uniapp.dcloud.io/collocation/pages?id=easycom)方式引入。

```js
// pages.json

{
    "easycom": {
        "^z-(.*)": "press-swiper/components/z-$1/z-$1.vue"
    },
    "pages": [...],
    "globalStyle": {
        ...
    }
}
```
